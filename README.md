# BlueOak Build System

The BlueOak build system is a comprehensive build system that integrates Browsersync, Cordova, and IBM MFP to reduce the amount of time spent on the table stakes of app development.

Take a look at [the wiki](https://github.com/BlueoakJS/generator-blueoak/wiki/BlueOak-Quickstart-Guide) to get started.
